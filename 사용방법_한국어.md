## 1. 소프트웨어 소개

**논문 읽기 도우미**는 로컬에서 실행되는 AI 논문 읽기 도구입니다.

- PDF 업로드 → AI가 자동으로 요약, 노트, 비판 분석 생성
- Claude · OpenAI · Gemini 세 가지 AI 서비스 지원
- 완전 로컬 실행 — PDF와 노트는 어떤 서버에도 업로드되지 않음

## 2. Gemini API Key 받기

> **Gemini 추천 이유**: 무료 한도가 충분하고, 신용카드 없이 사용 가능합니다. 신규 사용자는 분당 약 15회 호출 가능합니다.

### 단계별 안내

**① Google AI Studio 열기**

브라우저에서 접속: **https://aistudio.google.com**

---

**② Google 계정으로 로그인**

오른쪽 상단 「Sign in」 클릭 후, Google 계정으로 로그인합니다.

> Google 계정이 없으신가요? https://accounts.google.com/signup 에서 무료 가입하세요.

---

**③ API Key 생성**

로그인 후, 페이지 왼쪽 메뉴에서 **「Get API key」** 클릭

→ 파란색 버튼 **「Create API key」** 클릭

→ 대화 상자에서 「**Create API key in new project**」 선택

---

**④ API Key 복사**

화면에 `AIza`로 시작하는 키가 표시됩니다. 예시:

```
AIzaSyAbCdEfGhIjKlMnOpQrStUvWxYz1234567
```

키 옆의 **복사 아이콘（📋）** 클릭하거나 수동으로 전체 선택하여 복사합니다.

> ⚠️ **주의:** API Key는 한 번만 표시됩니다. 즉시 안전한 곳에 저장하세요（예: 메모앱）.

---

**⑤ 무료 한도 안내**

| 모델 | 무료 한도 | 추천 용도 |
|------|----------|---------|
| gemini-2.5-flash | 500회/일, 15회/분 | 일반 사용（추천）|
| gemini-2.5-pro | 25회/일 | 더 정확한 분석이 필요할 때 |

## 3. 소프트웨어 설치

### 시스템 요구사항

| 플랫폼 | 요구사항 |
|--------|---------|
| Mac | macOS 12 이상, Python 3.9+ |
| Windows | Windows 10/11, Python 3.9+ |
| 브라우저 | Chrome · Safari · Firefox · Edge（모두 가능）|

---

### Mac 설치

#### 1단계: Python 설치（이미 설치된 경우 건너뛰기）

「터미널」 열기（Spotlight에서 「Terminal」 검색）, 아래 입력:

```bash
python3 --version
```

`Python 3.x.x`가 표시되면 이미 설치된 것입니다. 없다면 공식 사이트에서 다운로드:

**https://www.python.org/downloads/macos/**

---

#### 2단계: 설치 스크립트 실행

`mac/install.command` **더블클릭**

처음 실행 시 macOS가 차단할 수 있습니다: 파일을 **우클릭 → 「열기」→ 팝업에서 「열기」** 클릭

터미널이 자동으로 열려 모든 의존성을 설치합니다（약 **2-5분**）. 「설치 완료」 메시지가 나오면 창을 닫으세요.

---

#### 3단계: .app으로 패키징

`mac/build.sh` **더블클릭**（마찬가지로 우클릭 → 열기로 보안 차단 처리）

패키징 완료 후 `dist/` 폴더에 생성됩니다:
- `논문 읽기 도우미.app` — 앱 파일
- `논문 읽기 도우미-Mac.dmg` — 배포용 설치 파일

`.app` 파일을 「응용 프로그램」 폴더로 드래그하면 설치 완료입니다.

---

### Windows 설치

#### 1단계: Python 설치（이미 설치된 경우 건너뛰기）

「명령 프롬프트」 열기（Win+R → `cmd` 입력）, 아래 입력:

```
python --version
```

찾을 수 없다는 메시지가 나오면:

1. **https://www.python.org/downloads/windows/** 접속
2. 최신 버전 설치 파일 다운로드（예: `python-3.12.x-amd64.exe`）
3. 설치 파일 실행 시 **「Add Python to PATH」 반드시 체크**
4. 설치 완료 후 명령 프롬프트를 다시 열어서 확인

---

#### 2단계: 설치 스크립트 실행

`windows\install.bat` **더블클릭**

명령 프롬프트 창이 열려 자동으로 의존성을 설치합니다（약 **2-5분**）. 「설치 완료」 메시지가 나오면 창을 닫으세요.

---

#### 3단계: .exe로 패키징

`windows\build.bat` **더블클릭**

패키징 완료 후 `dist\` 폴더에 생성됩니다:
- `PaperReader\PaperReader.exe` — 실행 파일
- `PaperReader-Windows.zip` — 배포용 압축 파일

zip 파일 압축 해제 후 `PaperReader.exe`를 더블클릭하면 실행됩니다.

## 4. 소프트웨어 실행

| 플랫폼 | 실행 방법 |
|--------|---------|
| Mac | 「논문 읽기 도우미.app」 더블클릭 |
| Windows | 「PaperReader.exe」 더블클릭 |

실행 후 작은 상태 창이 나타나며 「✓ 서비스 실행 중」이 표시되고, 브라우저가 **자동으로** 앱 화면을 엽니다.

- 「**브라우저 열기**」 버튼으로 언제든지 화면을 다시 열 수 있습니다
- 「**종료**」 버튼 또는 상태 창을 닫으면 앱이 완전히 종료됩니다

> 브라우저가 자동으로 열리지 않으면 `http://127.0.0.1:8000/` 을 직접 입력하세요.

## 5. API Key 설정

처음 실행 후 API Key를 입력해야 합니다:

1. 오른쪽 상단 **「⚙ 설정」** 클릭
2. **「AI 서비스」** 에서 **「Gemini (Google)」** 선택
3. **「API Key」** 입력란에 복사한 키 붙여넣기
4. **「검증」** 클릭하여 키 유효성 확인（✓ 표시되면 성공）
5. **「출력 언어」** 에서 원하는 분석 언어 선택
6. **「저장」** 클릭

> API Key는 브라우저 로컬에만 저장되며, 어떤 서버에도 업로드되지 않습니다.

## 6. 기능 설명

### 논문 열기

- 상단 바 「**논문 열기**」 버튼 클릭하여 PDF 파일 선택
- 또는 PDF 파일을 뷰어 영역으로 **드래그 앤 드롭**

### 분석 모드

「**분석**」 클릭 후 분석 목적 선택:

| 모드 | 적합한 경우 |
|------|-----------|
| 핵심 관점 빠르게 파악 | 논문의 핵심만 빠르게 파악, 정독하지 않을 때 |
| 정독 전 예비 탐색 | 정독 준비, 논리 구조 파악이 필요할 때 |
| 특정 질문이 있을 때 | 목적을 가지고 읽기, 질문 중심으로 전개 |
| 구조화된 정독 노트 | 핵심 문헌을 깊이 소화해야 할 때 |
| 비판적 분석 | 논문 신뢰성 평가, 심사 시뮬레이션 |
| 연구 확장 및 주제 탐색 | 자신의 연구 진입점 찾기 |

### 기타 기능

| 기능 | 조작 방법 |
|------|---------|
| PDF 내 검색 | `Ctrl+F`（Windows）/ `Cmd+F`（Mac）|
| 텍스트 복사 | 마우스로 텍스트 선택 → 「복사」 클릭 |
| 노트 추가 | 마우스로 텍스트 선택 → 「노트 추가」 클릭 |
| 노트 내보내기 | 사이드바 「↓ 내보내기」→ Markdown 파일 |
| PDF → Markdown | 상단 바 「PDF → MD」 버튼 |
| 서재 | 논문 및 분석 결과 저장, 언제든 다시 불러오기 |
| 인터페이스 언어 | 상단 바 드롭다운 메뉴, 9개 언어 지원 |
| 다중 탭 | 상단 바 「＋」 버튼, 동시에 여러 논문 열기 |

## 7. 자주 묻는 질문

**Q: Mac에서 「Apple이 악성 소프트웨어를 확인할 수 없어 열 수 없습니다」라고 나올 때?**

A: `.app` 파일을 **우클릭 → 「열기」→ 팝업에서 「열기」** 클릭하세요. 이 메시지는 처음 한 번만 나타나며, 이후에는 더블클릭으로 바로 실행됩니다.

---

**Q: 설치 중 「pip 네트워크 타임아웃」이 발생할 때?**

A: pip 미러를 변경한 후 다시 시도하세요:

```bash
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```

---

**Q: Windows에서 「Python을 찾을 수 없음」인데 이미 설치했을 때?**

A: Python을 다시 설치하고, 설치 시 하단의 **「Add Python to PATH」를 반드시 체크**한 후, 명령 프롬프트를 **다시 열어서** 확인하세요.

---

**Q: 브라우저에서 「사이트에 연결할 수 없음」이 표시될 때?**

A: 약 10초 정도 기다렸다가 페이지를 새로고침하세요. 그래도 안 되면 상태 창을 닫고 앱을 다시 실행하세요.

---

**Q: API Key 입력 후 「검증 실패」가 표시될 때?**

A: 다음을 확인하세요:
1. 키가 완전히 복사되었는지（불필요한 공백 없이）
2. Gemini Key는 `AIza`로 시작하는지
3. AI Studio에서 계정이 활성화되었는지

---

**Q: 분석 결과가 영어로 나올 때 한국어로 변경하려면?**

A: 설정 → 「출력 언어」→ 「한국어」 선택 → 저장 → 다시 분석.

---

**Q: 완전히 제거하려면?**

A: `paper-reader` 폴더（또는 `.app` 파일）를 삭제하면 됩니다. 시스템 레지스트리 수정이 없습니다.
