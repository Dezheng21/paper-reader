## 1. 软件简介 · 소프트웨어 소개

**论文阅读助手** 是一款本地运行的 AI 论文阅读工具。  
**논문 읽기 도우미**는 로컬에서 실행되는 AI 논문 읽기 도구입니다.

- 上传 PDF → AI 自动生成导读、笔记、批判分析
- PDF 업로드 → AI가 자동으로 요약, 노트, 비판 분석 생성
- 支持 Claude · OpenAI · Gemini 三大 AI 服务商
- Claude · OpenAI · Gemini 세 가지 AI 서비스 지원
- 全程本地运行，PDF 和笔记不上传至任何服务器
- 완전 로컬 실행, PDF와 노트는 어떤 서버에도 업로드되지 않음

## 3. 安装软件 · 소프트웨어 설치

### 系统要求 · 시스템 요구사항

| 平台 | 要求 |
|------|------|
| Mac | macOS 12 或以上，Python 3.9+ |
| Windows | Windows 10/11，Python 3.9+ |
| 浏览器 | Chrome · Safari · Firefox · Edge（任一） |

---

### Mac 安装

#### 第一步：安装 Python（如已安装可跳过）

打开「终端」（Spotlight 搜索「Terminal」），输入：

```bash
python3 --version
```

若显示 `Python 3.x.x` 即已安装。若提示未找到，运行：

```bash
# 方法一：官网下载（推荐新手）
# 访问 https://www.python.org/downloads/macos/ 下载安装包

# 方法二：Homebrew
brew install python
```

---

#### 第二步：给脚本授予执行权限

首次使用时，macOS 会阻止未签名脚本。  
在「终端」中执行（将路径替换为你的实际路径）：

```bash
chmod +x /你的路径/paper-reader/mac/install.command
chmod +x /你的路径/paper-reader/mac/run.command
```

或者右键点击文件 → **「打开方式」→「终端」**，在弹出的安全提示中点击「打开」。

---

#### 第三步：运行安装脚本

**双击** `mac/install.command`

终端窗口会自动打开并安装所有依赖。安装过程约 **2-5 分钟**（视网络速度）。  
출력 예시：

```
▶  检查 Python 版本…
✔  Python 3.12  (arm64)
▶  创建虚拟环境 .venv …
✔  虚拟环境创建完成
▶  安装依赖包（首次约 2-5 分钟）…
✔  所有依赖已安装
✅ 安装完成！
```

看到「安装完成」即可关闭此窗口。

---

### Windows 安装

#### 第一步：安装 Python（如已安装可跳过）

打开「命令提示符」（Win+R 输入 `cmd`），输入：

```
python --version
```

若显示 `Python 3.x.x` 即已安装。若提示未找到：

1. 访问 **https://www.python.org/downloads/windows/**
2. 下载最新版本安装包（如 `python-3.12.x-amd64.exe`）
3. 运行安装包，**务必勾选「Add Python to PATH」**
4. 完成安装后重新打开命令提示符验证

---

#### 第二步：运行安装脚本

**双击** `windows\install.bat`

命令提示符窗口会打开并自动安装所有依赖。  
출력 예시：

```
══════════════════════════════════════════
   论文阅读助手  —  Windows 首次安装
══════════════════════════════════════════

[✔] Python 3.12.0
[▶] 创建虚拟环境 .venv ...
[✔] 虚拟环境创建完成
[▶] 安装依赖包（首次约 2-5 分钟）...
[✔] 所有依赖已安装
✅ 安装完成！
```

## 4. 启动软件 · 소프트웨어 실행

安装完成后，每次使用只需：  
설치 완료 후, 매번 사용 시：

| 平台 | 操作 |
|------|------|
| Mac | 双击 `mac/run.command` |
| Windows | 双击 `windows\run.bat` |

浏览器会**自动打开** `http://127.0.0.1:8000/`  
브라우저가 **자동으로** `http://127.0.0.1:8000/` 를 엽니다

> 若浏览器没有自动打开，手动在浏览器地址栏输入 `http://127.0.0.1:8000/`  
> 브라우저가 자동으로 열리지 않으면, 주소창에 `http://127.0.0.1:8000/` 직접 입력

## 5. 配置 API Key · API Key 설정

启动后首次使用，需要填入 API Key：  
처음 시작 후 API Key를 입력해야 합니다：

1. 点击右上角 **「⚙ 设置」**  
   오른쪽 상단 **「⚙ 설정」** 클릭

2. **「AI 服务商」** 选择 **「Gemini (Google)」**  
   **「AI 서비스」** 에서 **「Gemini (Google)」** 선택

3. 在 **「API Key」** 框中粘贴你复制的密钥  
   **「API Key」** 입력란에 복사한 키 붙여넣기

4. 点击 **「验证」** 确认密钥有效（显示 ✓ 表示成功）  
   **「검증」** 클릭하여 키 유효성 확인（✓ 표시되면 성공）

5. **「输出语言」** 选择你想要的分析语言  
   **「출력 언어」** 에서 원하는 분석 언어 선택

6. 点击 **「保存」**  
   **「저장」** 클릭

> API Key 仅保存在你的浏览器本地，不会上传到任何服务器。  
> API Key는 브라우저 로컬에만 저장되며, 어떤 서버에도 업로드되지 않습니다.

## 6. 功能说明 · 기능 설명

### 打开论文 · 논문 열기

- 点击顶栏「**打开论文**」按钮选择 PDF 文件
- 또는 PDF 파일을 뷰어 영역으로 **드래그 앤 드롭**

### 分析模式 · 분석 모드

点击「**分析结构**」后选择分析目的：  
「**분석**」클릭 후 분석 목적 선택：

| 模式 | 적합한 경우 |
|------|------------|
| 快速了解核心观点 | 快速获取论文主旨，不打算精读 |
| 精读前的预热导航 | 准备精读，需要掌握逻辑框架 |
| 我有一个具体问题 | 带着目的读，围绕你的问题展开 |
| 结构化精读笔记 | 需要深度消化核心文献 |
| 批判性分析 | 评估论文可信度，模拟审稿视角 |
| 延伸研究与选题 | 为自己的研究找切入点 |

### 其他功能 · 기타 기능

| 功能 | 操作 |
|------|------|
| PDF 内搜索 | `Ctrl+F`（Windows）/ `Cmd+F`（Mac）|
| 文本复制 | 鼠标选中文字 → 点击「复制」|
| 添加笔记 | 鼠标选中文字 → 点击「添加笔记」|
| 导出笔记 | 侧栏「↓ 导出」→ Markdown 文件 |
| PDF → Markdown | 顶栏「PDF → MD」按钮 |
| 书架 | 保存论文和分析结果，随时重新加载 |
| 界面语言 | 顶栏下拉菜单，支持 9 种语言 |
| 多标签页 | 顶栏「＋」按钮，同时打开多篇论文 |

## 7. 常见问题 · 자주 묻는 질문

**Q: 安装时出现「pip 网络超时」？**  
A: 更换 pip 镜像源后重试：
```bash
# Mac / Linux
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

# Windows（在命令提示符中）
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```

---

**Q: Mac 提示「无法打开，因为 Apple 无法检查是否包含恶意软件」？**  
A: 右键点击 `.command` 文件 → **「打开」** → 在弹框中点「打开」即可。  
或在终端执行：`xattr -cr mac/install.command mac/run.command`

---

**Q: Windows 提示「Python 未找到」但已安装 Python？**  
A: 重新安装 Python，安装时务必勾选底部的「**Add Python to PATH**」选项，然后**重新打开**命令提示符。

---

**Q: 浏览器打开后显示「无法访问此网站」？**  
A: 等待约 10 秒让服务完全启动，然后手动刷新页面。若仍无法访问，关闭启动窗口，重新双击启动脚本。

---

**Q: API Key 填入后显示「验证失败」？**  
A: 检查以下几点：
1. 复制时是否完整（不含多余空格）
2. Gemini Key 以 `AIza` 开头
3. 在 AI Studio 中确认账号已激活

---

**Q: 分析结果是英文，如何改为中文？**  
A: 设置 → 「输出语言」→ 选择「中文（简体）」→ 保存 → 重新分析。

---

**Q: 如何完全卸载？**  
A: 直接删除整个 `paper-reader` 文件夹即可，无系统级注册表修改。  
설치 폴더 전체를 삭제하면 됩니다. 시스템 레지스트리 수정이 없습니다.
