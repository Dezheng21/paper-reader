## 1. 软件简介

**PaperKnowKnow** 是一款本地运行的 AI 论文阅读工具。

- 上传 PDF，AI 自动生成导读、笔记、批判分析
- 支持 Claude · OpenAI · Gemini 三大 AI 服务商
- 全程本地运行，PDF 和笔记不上传至任何服务器

## 2. 获取 Gemini API Key

> **推荐使用 Gemini**：免费额度充足，无需信用卡，新用户每分钟可调用约 15 次。

### 操作步骤

**① 打开 Google AI Studio**

用浏览器访问：**https://aistudio.google.com**

---

**② 登录 Google 账号**

点击右上角「Sign in」，用你的 Google 账号登录。

> 没有 Google 账号？访问 https://accounts.google.com/signup 免费注册。

---

**③ 创建 API Key**

登录后，在页面左侧菜单点击 **「Get API key」**

→ 点击蓝色按钮 **「Create API key」**

→ 在弹出的对话框中，选择「**Create API key in new project**」

---

**④ 复制 API Key**

屏幕上会显示一串以 `AIza` 开头的密钥，例如：

```

```

点击密钥旁边的 **复制图标（📋）** 或手动全选复制。

> ⚠️ **注意：** API Key 只显示一次，请立即保存到安全位置（如备忘录）。

## 3. 安装软件

### 系统要求

| 平台 | 要求 |
|------|------|
| Mac | macOS 12 或以上，Python 3.9+ |
| Windows | Windows 10/11，Python 3.9+ |
| 浏览器 | Chrome · Safari · Firefox · Edge（任一）|

---

### Mac 安装

#### 第一步：安装 Python（已安装可跳过）

打开「终端」（Spotlight 搜索「Terminal」），输入：

```bash
python3 --version
```

若显示 `Python 3.x.x` 即已安装。若提示未找到，前往官网下载：

**https://www.python.org/downloads/macos/**

---

#### 第二步：运行安装脚本

**双击** `mac/install.command`

首次运行时 macOS 可能拦截，请：右键点击文件 → **「打开」** → 弹框中点「打开」

终端会自动安装所有依赖，约 **2-5 分钟**（视网络速度）。出现「安装完成」即可关闭窗口。

---

#### 第三步：打包为 .app

**双击** `mac/build.sh`（同样右键→打开处理安全拦截）

打包完成后，`dist/` 文件夹中会生成：
- `PaperKnowKnow.app` — 应用程序
- `PaperKnowKnow-Mac.dmg` — 可分发的安装包

将 `.app` 拖入「应用程序」文件夹即完成安装。

---

### Windows 安装

#### 第一步：安装 Python（已安装可跳过）

打开「命令提示符」（Win+R 输入 `cmd`），输入：

```
python --version
```

若提示未找到：

1. 访问 **https://www.python.org/downloads/windows/**
2. 下载最新版安装包（如 `python-3.12.x-amd64.exe`）
3. 运行安装包，**务必勾选「Add Python to PATH」**
4. 安装完成后重新打开命令提示符验证

---

#### 第二步：运行安装脚本（仅需一次）

**双击** `windows\install.bat`

命令提示符窗口会自动完成以下操作，约 **2-5 分钟**：
- 自动安装 Python（如系统中没有）
- 安装所有依赖包
- 在**桌面**生成「PaperKnowKnow」快捷方式

出现「Installation complete」即可关闭窗口。

## 4. 启动软件

| 平台 | 操作 |
|------|------|
| Mac | 双击「PaperKnowKnow.app」|
| Windows | 双击桌面上的「PaperKnowKnow」快捷方式 |

**Windows** 启动后会出现一个命令行窗口（服务运行期间保持开启），浏览器会**自动打开**应用界面。关闭命令行窗口即退出软件。

**Mac** 启动后浏览器自动打开，Dock 栏显示应用图标。

> 若浏览器未自动打开，手动访问 `http://127.0.0.1:8000/`

## 5. 配置 API Key

首次启动后，需要填入 API Key：

1. 点击右上角 **「⚙ 设置」**
2. **「AI 服务商」** 选择 **「Gemini (Google)」**
3. 在 **「API Key」** 框中粘贴你复制的密钥
4. 点击 **「验证」** 确认密钥有效（显示 ✓ 表示成功）
5. **「输出语言」** 选择你想要的分析语言
6. 点击 **「保存」**

> API Key 仅保存在你的浏览器本地，不会上传到任何服务器。

## 6. 功能说明

### 打开论文

- 点击顶栏「**打开论文**」按钮选择 PDF 文件
- 或将 PDF 文件直接拖放到查看区域

### 分析模式

点击「**分析**」后选择分析目的：

| 模式 | 适用场景 |
|------|---------|
| 快速了解核心观点 | 快速获取论文主旨，不打算精读 |
| 精读前的预热导航 | 准备精读，需要掌握逻辑框架 |
| 我有一个具体问题 | 带着目的读，围绕你的问题展开 |
| 结构化精读笔记 | 需要深度消化核心文献 |
| 批判性分析 | 评估论文可信度，模拟审稿视角 |
| 延伸研究与选题 | 为自己的研究找切入点 |

### 其他功能

| 功能 | 操作 |
|------|------|
| PDF 内搜索 | `Ctrl+F`（Windows）/ `Cmd+F`（Mac）|
| 文本复制 | 鼠标选中文字 → 点击「复制」|
| 添加笔记 | 鼠标选中文字 → 点击「添加笔记」|
| 导出笔记 | 侧栏「↓ 导出」→ Markdown 文件 |
| PDF → Markdown | 顶栏「PDF → MD」按钮 |
| 书架 | 保存论文和分析结果，随时重新加载 |
| 界面语言 | 顶栏下拉菜单，支持 9 种语言 |
| 多标签页 | 顶栏「＋」按钮，同时打开多篇论文 |

## 7. 常见问题

**Q: Mac 提示「无法打开，因为 Apple 无法检查是否包含恶意软件」？**

A: 右键点击 `.app` → **「打开」** → 在弹框中点「打开」即可。此提示仅出现一次，之后可直接双击启动。

---

**Q: 安装时出现「pip 网络超时」？**

A: 更换 pip 镜像源后重试：

```bash
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```

---

**Q: Windows 提示「Python 未找到」但已安装 Python？**

A: 重新安装 Python，安装时务必勾选底部的「**Add Python to PATH**」选项，然后**重新打开**命令提示符。

---

**Q: 浏览器打开后显示「无法访问此网站」？**

A: 等待约 10 秒让服务完全启动，然后手动刷新页面。若仍无法访问，关闭状态窗口，重新双击启动。

---

**Q: API Key 填入后显示「验证失败」？**

A: 检查以下几点：
1. 复制时是否完整（不含多余空格）
2. Gemini Key 以 `AIza` 开头
3. 在 AI Studio 中确认账号已激活

---

**Q: 分析结果是英文，如何改为中文？**

A: 设置 → 「输出语言」→ 选择「中文（简体）」→ 保存 → 重新分析。

---

**Q: 如何完全卸载？**

A: 直接删除整个 `PaperKnowKnow` 文件夹（或 `.app` 文件）即可，无系统级注册表修改。
